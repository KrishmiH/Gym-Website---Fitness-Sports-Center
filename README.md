## Project Overview

This project was built for a local fitness brand launching a new gym. The goal is to attract new members using a modern, responsive landing experience that introduces services, trainers, plans, and an inquiry workflow.

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Framer Motion
- React Hook Form

## Implemented Home Page Sections

- Hero Section
- About Section
- Features / Services Section
- Membership Plans Section
- Trainers Section
- Contact Section
- Header + Footer

## Contact Form

- Fields: Name, Email, Message
- Client-side validation
- Inline field-level error messages
- Success and error feedback on submit
- Character counter and submission loading state

## Features Implemented

- Dark mode with preference persistence
- Search and filter functionality (Services section)
- Scroll-reveal animations
- API integration (form submit)
- Advanced form handling (React Hook Form + Zod)

## Local Setup

1. Install dependencies

```bash
npm install
```

2. Start development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Preview production build

```bash
npm run preview
```

## Deployment (Vercel)

The project was deployed with Vercel CLI:

```bash
npx vercel login
npx vercel --prod --yes
npx vercel alias set gym-website-3epdsfnvx-krishmihs-projects.vercel.app gym-website-fitness-sports-center.vercel.app
```
