# FitForge Gym Promotional Website

Responsive promotional website developed for the Web Development Internship Evaluation.

## Tech Stack

- React 19
- Vite 8
- CSS3
- ESLint

## Implemented Sections

- Hero Section
- About Section
- Features / Services Section
- Membership Plans Section
- Trainers Section
- Contact Section

## Contact Form Features

- Fields: Name, Email, Message
- Client-side validation
- Inline error messages
- Success feedback after submission

## Setup Instructions

1. Install dependencies

	```bash
	npm install
	```

2. Run development server

	```bash
	npm run dev
	```

3. Build production bundle

	```bash
	npm run build
	```

4. Preview production build

	```bash
	npm run preview
	```

Place your final images from the provided branding set in `src/assets` and update section image references as needed.
